Customer Shopping Behavior Analysis 🛍️
- An end-to-end data analysis project on 3,900 customer transactions, covering the full pipeline from raw data to interactive dashboard.

1) EDA in Python — Cleaned data using pandas, handled 37 missing values in Review Rating via category-wise median imputation, engineered age_group and purchase_frequency_days features and loaded the cleaned data into MySQL.

2) SQL Analysis — Wrote 10 business queries in MySQL covering revenue by gender & age group, customer segmentation (New / Returning / Loyal), top products by rating and category, discount behavior and subscriber vs non-subscriber spend.

3) Power BI Dashboard — Built an interactive dashboard with slicers for gender, category, subscription status and shipping type, visualizing KPIs like average purchase amount, revenue by category, and age group trends.
